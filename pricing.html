✦ Questions fréquentes ✦

Tout ce que vous voulez savoir

Les scénarios sont-ils vraiment uniques ?▼
Oui. Notre moteur génère les personnages, leurs secrets, leurs alibis, les indices et la solution en combinant des milliers de variables. Même en choisissant les mêmes paramètres, vous obtiendrez un scénario différent à chaque fois.
L'IA a-t-elle besoin d'une connexion internet ?▼
Non. Le moteur de génération fonctionne entièrement dans votre navigateur, sans aucun appel à une API externe. Une connexion est uniquement nécessaire pour charger la page la première fois (Google Fonts).
Que contient exactement le PDF ?▼
Le PDF comprend : le texte d'introduction narratif, une fiche A5 par personnage (portrait, personnalité, mobile, alibi, secret, objet distinctif), les cartes indices à découper, la chronologie visuelle, et le guide complet du Maître de Jeu avec la révélation et le chemin de déduction.
Puis-je utiliser MysteryForge pour des événements commerciaux ?▼
Le plan Organisateur est fait pour ça. Il inclut l'export marque blanche (votre logo sur les PDF), des tarifs dégressifs pour les volumes et une API pour intégrer le générateur à votre propre plateforme.
Comment annuler mon abonnement ?▼
L'annulation se fait en un clic depuis votre espace client. Aucun préavis requis. Vous conservez l'accès jusqu'à la fin de la période déjà payée.
Puis-je modifier le scénario après génération ?▼
Dans la version gratuite et Détective, vous pouvez copier le texte et le modifier librement. Le plan Organisateur inclut un éditeur intégré pour modifier noms, descriptions et indices directement avant l'export PDF.
