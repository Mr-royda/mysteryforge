Questions fréquentes

C'est quoi exactement une soirée murder mystery ?▼
Un jeu d'enquête en immersion où chaque invité joue un personnage. Un crime fictif a eu lieu — tout le monde interroge, accuse, démasque. À la fin, le Maître de Jeu révèle la vérité. C'est un croisement entre jeu de rôle, escape game et soirée déguisée. Aucune expérience préalable nécessaire — le scénario guide tout le monde.
Dois-je avoir des compétences pour organiser la soirée ?▼
Non. Le guide du Maître de Jeu explique tout : comment introduire le crime, quand distribuer les indices, comment relancer l'enquête si elle s'embourbe, et comment conduire la révélation finale. Même si c'est votre première fois, vous êtes guidé étape par étape.
Les scénarios sont-ils vraiment uniques à chaque génération ?▼
Oui. Notre moteur combine des milliers de variables : noms, portraits, personnalités, mobiles, alibis, méthodes de meurtre, indices, fausses pistes. Même en choisissant exactement les mêmes paramètres, vous obtiendrez un scénario différent à chaque fois. Aucune API externe n'est utilisée — tout se génère dans votre navigateur.
Que contient le PDF téléchargeable ?▼
Le PDF comprend : le texte d'introduction narratif, une fiche A5 par joueur (portrait, personnalité, mobile, alibi, secret, objet distinctif), les cartes indices à découper, la chronologie visuelle, et le guide complet du Maître de Jeu avec révélation, chemin de déduction et conseils d'animation.
Puis-je annuler à tout moment ?▼
Oui, à tout moment depuis votre espace client, en un clic. Aucun préavis requis. Vous conservez l'accès jusqu'à la fin de la période déjà payée. Et si vous n'êtes pas satisfait dans les 7 premiers jours, nous vous remboursons intégralement.
Puis-je utiliser MysteryForge pour des événements professionnels ?▼
Le plan Organisateur Pro est fait pour ça : PDF marque blanche avec votre logo, scénarios pour jusqu'à 20 joueurs, éditeur de personnages, API REST pour intégrer à votre propre plateforme. Contactez-nous pour les tarifs sur volume.
Mon paiement est-il sécurisé ?▼
Oui. Les paiements sont traités par Stripe, leader mondial du paiement en ligne (utilisé par Amazon, Google, Airbnb…). Vos coordonnées bancaires ne transitent jamais par nos serveurs. La connexion est chiffrée SSL de bout en bout.
